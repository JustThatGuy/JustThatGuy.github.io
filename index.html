Profielwerkstuk

Robin Potze | 2017

Dit is mijn profielwerkstuk, het project dat elke niet technasiumstudent moet maken in hun examenjaar. Ik heb het verslag verwerkt in het project zelf, een website met een parallax-scrolling systeem voor het vak informatica. Ik wil dit graag persoonlijk houden, maar ik denk dat ik daar op benadeeld wordt. Mijn persoonlijkheid laat ik dan wel achter in deze inleiding denk ik. Hoop ik.

Plan van aanpak

Heading for Portion 2

Ik begin met het onderzoeken van html en het leren van de talen, hierna bedenk ik waar de site over moet gaan en dan verzin ik een mooi ontwerp bij het onderwerp en het doel van de site. Dit ga ik ontwerpen doormiddel van css en jquery. Het verslag schrijf ik nadat de site staat.
